	- Filter Row
	- Column Layout
	- Star Sizing
	- CellFreezing & Merging
	- Custom Merging
	- Unbound
	- On Demand
	- Custom Appearance
	- New Row
	- Check-List
	- Live Updates
	- Custom Sort Icon
	- Export
	- DataTable Sample
